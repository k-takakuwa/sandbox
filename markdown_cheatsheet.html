Markdown 構文チートシート

Markdownの基本的な構文をまとめたリファレンスです。

見出し (Headings)

#の数でレベルが変わります。#の後に半角スペースを入れます。

# 見出し1
## 見出し2
### 見出し3
#### 見出し4

見出し1

見出し2

見出し3

見出し4

強調 (Emphasis)

太字は**または__、斜体は*または_で囲みます。

**太字のテキスト**
_斜体のテキスト_
***太字と斜体のテキスト***

太字のテキスト
斜体のテキスト
太字と斜体のテキスト

箇条書き (Lists)

順序なしリストは*、順序付きリストは1.から始めます。

* アイテム1
* アイテム2
    - サブアイテムa
1. 最初のアイテム
2. 次のアイテム
  • アイテム1
  • アイテム2
    • サブアイテムa
  1. 最初のアイテム
  2. 次のアイテム

リンク (Links)

[表示するテキスト](URL)の形式で書きます。

[Googleのウェブサイト](https://www.google.com)

Googleのウェブサイト

![代替テキスト](画像URL)の形式で、リンクと似ています。

コード (Code)

インラインは`で、ブロックは```で囲みます。

`git commit` コマンドを使います。

```python
def hello_world():
    print("Hello, World!")
```

    print("Hello, World!")

引用 (Blockquotes)

行頭に>を付けます。

> これは引用文です。
> 別の行も同じブロックに含められます。
これは引用文です。
別の行も同じブロックに含められます。

水平線 (Horizontal Rules)

***、---、___ を単独の行に書きます。

テキスト
***
別のテキスト

テキスト

別のテキスト

表 (Tables)

ヘッダーとセルの区切りを|で、ヘッダーとボディの区切りを---で表現します。

| ヘッダー1 | ヘッダー2 | ヘッダー3 |
| :--- | :---: | ---: |
| 左寄せ | 中央寄せ | 右寄せ |
| 値1 | 値2 | 値3 |
